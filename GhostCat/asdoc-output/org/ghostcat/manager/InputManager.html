Class InputManagerProperties | Methods
 
Packageorg.ghostcat.manager
Classpublic class InputManager
InheritanceInputManager Singleton flash.events.EventDispatcher

输入设备管理类，附加功能为： 检测暂离，检测按钮是否按下，检测单独按键的一次点击（非组合键），鼠标手势，鼠标右键，连续按键 实现方式是向外部发布InputEvent事件 启用鼠标右键时，为了禁止FLASH自己的右键菜单，必须将FLASH的wmode改为opaque
