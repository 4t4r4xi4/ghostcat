public static function eval(value:String, root:Object = null):*

通过点操作符查找属性 显示对象的[]操作符可以用来查询子对象 诸如eval("items[0].panel.buttonBar[2].text",root)

Parameters
value:String — 路径
 
root:Object (default = null) — 起始对象 不设置此参数的话，将会自动反射路径的第一段。因此唯一合法的写法是让第一段成为一个类，第二段则是这个类的某个静态属性。 诸如：eval("Queue.defaultQueue.commit")();
